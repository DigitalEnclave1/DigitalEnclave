# DigitalEnclave – Modern IT Solutions

<img src="/public/site.png" alt="DigitalEnclave Screenshot" width="100%">

**DigitalEnclave** is a modern IT company website built with **Next.js (App Router), TypeScript, React, Tailwind CSS, and Framer Motion**.

## Tech Stack

- **Framework:** [Next.js (App Router)](https://nextjs.org/docs)  
- **Language:** TypeScript  
- **UI Library:** React  
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)  
- **Animations:** [Framer Motion](https://www.framer.com/motion/)    

---

## Features

- **Modern UI/UX:** Sleek and professional design with smooth animations  
- **Responsive Design:** Fully functional across devices and screen sizes

## Installation & Setup

1. **Clone the repository**
   ```bash
    git clone https://github.com/shashankneupane0427/DigitalEnclave.git
    cd DigitalEnclave
   ```
2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```
3. **Run the development server**
   ```bash
   npm run dev
   # or
   yarn dev
   ```
